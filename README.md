# React Starter App

## Package Support Included
 - Webpack 3.8
 - React 17.0
 - React Router 6.0
 - Typescript 4.5
 - Less
 - Hygen 5.0
   - `hygen component new` to create new components.
 - Unstateless 0.14
   - `inject` boilerplate included in new Hygen components.
 - AWS S3
   - `npm run deploy` to build and deploy the app to a specified S3 bucket.
